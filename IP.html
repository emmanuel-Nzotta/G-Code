Society has changed the world:

Network Control Program:

There were some data transmission protocols developed by ARPSNET. Network Control Program (NCP) was a program that was used to communicate between computers.

Transmission Control Protocol:

It was the birth of the TCP (Transmission Control Protocol) and also the IP (Internet Protocol). The inventor for the TCP and IP were Vinton Cerf and Bob Kahn. The TCP and IP is part of the groundwork for the modern-day internet. TCP is known for the reliability of data delivery through communication. IP is known for providing the route and addressing for data to be sent through different available networks.

Simple Mail Transfer Protocol:

SMTP (Simple Mail Transfer Protocol): Jon Postal and Suzanne Sulzer invented the SMPT in 1982. This protocol is used for receiving and sending email between servers. This plays an important role in the growth of email communication.

Domain Name System:

DNS (Domain Name System): It translates domain names into IP codes. This is used for navigating the internet.

Internet Protocol version 6:

IPv6 (Internet Protocol version 6): It gives the user a larger address space and improved security.

File Transfer Protocol:

FTP (File Transfer Protocol): This protocol is used to transfer files between computers. It is used for uploading and or downloading files. The downloading can come from a server or can go to a server.

Secure Shell Protocol:

SHH (Secure Shell Protocol): It provides access to servers and used to secure data communication. It is often used for remote administration.
